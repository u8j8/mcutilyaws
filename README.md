# mcutilya ws
## MCutilyaとは、tohonikiの作成したutilツール(utilya)系統のユーティルツールです
### 実行方法
1. `npm i` を実行
2. `npm run run`又は`node main.js`を実行（るんるん（笑））
## 機能
1. マインクラフトからチャット、入出退出を表示
2. 上記の物をdiscordに転送
3. コンソールからコマンドを実行
